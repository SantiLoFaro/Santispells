# Santispells

Tracker statico per slot incantesimo D&D 5e.

Funzioni:

- salvataggio automatico in `localStorage`
- selettore classe con tema di sfondo
- slot modificabili per livelli 1-9
- controlli per consumare, ripristinare o svuotare slot
- sezione Pact Magic per warlock

Pubblicazione: carica `index.html`, `styles.css`, `app.js` e questo README in un repository GitHub Pages.
